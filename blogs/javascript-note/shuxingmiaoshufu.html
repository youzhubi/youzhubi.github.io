属性描述符

fyj2023/05/15javascript-noteJavaScript

描述符是什么？
描述符（Descriptor）是用于定义和控制对象属性行为的对象。 每个对象属性都有一个相关联的描述符，描述符包含了一些属性的特性和行为。

# 1.得到属性描述符

const obj = {
    a: 1,
    b: 2,
}
const desc = Object.getOwnPropertyDescriptor(obj, "a")
console.log(desc)

# 2.设置属性描述符

Object.defineProperty(obj, "a", {
    value: 5,
    writable: false, // 不可重写
    enumerable: false, // 不可枚举（遍历）
    configurable: false, // 不可修改描述符本身
})

属性描述符中的getter和setter

const obj = {}
let internValue = undefined
Object.defineProperty(obj, "a", {
    // 读取器 getter
    get: function () {
        return internValue
    },
    // 设置器 setter
    set: function (val) {
        internValue = val
    },
})

TIP

  • writable、enumerable、configurable默认值为false。
  • value、get、set默认值为undefined。
  • getter和setter是一个隐藏函数，分别在获取属性值和设置属性值时被调用。
  • 一旦设置了getter和setter，该属性的value和writable则自动忽略。

# 3.Object.seal()（密封对象）

// Object.seal()密封对象
const obj = {
    name: "www",
    age: 23,
    sex: "女",
}
Object.seal(obj)
// 输出false
console.log(delete obj.name)
// 输出{name:'www',age: 23,sex:'女'},name属性依然存在
console.log(obj)
// address属性添加失败,严格模式下报错TypeError
obj.address = "广州"
// 输出{name:'www',age: 23,sex:'女'},address属性添加失败
console.log(obj)
// 报错,TypeError
Object.defineProperty(obj, "name", {
    configurable: true,
    writable: true,
    enumerable: true,
})

TIP

  • 对象密封后，不能添加新的属性，不能重新配置也不能删除现有属性。
  • 可以修改现有属性的值。
  • Object.seal()依次调用对象属性的Object.preventExtensions()方法。

# 4.Object.freeze()（冻结对象）

// Object.freeze()冻结对象
var obj = {
    name: "孙悟空",
}
Object.freeze(obj)
// 赋值失败，严格模式下，报错TypeError
obj.name = "沙和尚"
// 输出{name:'孙悟空'}
console.log(obj)
// 报错TypeError
Object.defineProperty(obj, "name", {
    value: "猪八戒",
})

TIP

  • 对象冻结后，不能添加新的属性，不能修改对象属性上的值，不能删除属性，不能修改属性描述符
  • 冻结一个对象后该对象的原型也不能被修改。
  • Object.freeze() 调用Object.seal()并把所有属性的writable修改为false。
  • 这是一个浅冻结，如果对象里还有对象，则对象的对象不收冻结影响。
