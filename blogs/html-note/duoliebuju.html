# 多栏布局（Multi-Columns Layout）

# 1.column-count

创建指定数量的列 — 由浏览器计算出每一列分配多少空间

 column-count: 3;

# 2..column-width

指定每列的宽度，浏览器会尽可能多的创建列。

column-width: 200px;

# 3.column-gap和column-rule

  • column-gap：指定每列之间的间隙
  • column-rule：在每列之间加入分割线
.container {
            column-count: 3;
            column-gap: 20px;
            column-rule: 4px dotted rgb(79, 185, 227);
}

# 4.列与内容折断

可以通过设置break-inside解决这种问题

break-inside: avoid;

# 5.columns

column-count 和 column-width的属性简写，column-count的优先级高于column-width

/* Column width */
columns: 18em;

/* Column count */
columns: auto;
columns: 2;

/* Both column width and count */
columns: 2 auto;
columns: auto 12em;
columns: auto auto;

/* Global values */
columns: inherit;
columns: initial;
columns: unset;

# 6.column-span

column-span设置为all，可以让一个元素跨所有列显示，可以用于文章插入图片通栏显示。

column-span: none;
column-span: all;

column-span: inherit;
